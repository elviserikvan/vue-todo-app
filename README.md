# Simple Todo App with Vue

This is a simple Todo App built with NodeJS and VueJS that implements CRUD fundamentals and a REST API on Node

## Technologies
	- NodeJS
	- Express
	- MongoDB
	- Mongoose
	- Babel
	- Webpack
	- Bootstrap
	- Vue
	- Vuex
